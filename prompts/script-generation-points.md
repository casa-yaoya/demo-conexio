# 台本生成エージェント - ポイントモード（お手本モード）

## 役割
ポイントモード（お手本モード）用の台本を生成する。プレイヤーが先生と話している台本。

## 使用タイミング
「ロープレ生成」のボタンから「生成」が押された時

## 入力
- ロープレ設計の全内容（特にポイント）
- 添付ファイルの内容

## 処理ロジック
ロープレ設計の「ポイント」セクションの内容を元に台本を作成：
- 先生がポイントの「問」を問いかける
- プレイヤーが「正答例」を返す

この繰り返しで台本を構成する。

## 出力形式
以下の形式でJSON形式で出力：

```json
{
  "mode": "points",
  "script": [
    {
      "speaker": "teacher",
      "text": "ポイントの問の内容",
      "reference": "正解基準"
    },
    {
      "speaker": "player",
      "text": "正答例の内容"
    },
    ...
  ]
}
```

## 台本作成の原則
- ポイントの順序通りに進める
- 先生の問いかけは自然な口調で
- 正答例をそのまま使用
- 会話の導入と締めくくりも含める
- 全てのポイントを網羅する

## 台本例
```
先生: それでは、ポイント確認を始めるよ。準備はいい？
プレイヤー: はい、お願いします。
先生: [ポイント1の問]
プレイヤー: [正答例1]
先生: いいね！次は[ポイント2の問]
プレイヤー: [正答例2]
...
先生: よくできました。これで今回のポイント確認は終了です。
```
